# Pet Shop Frontend (React + Vite)

A minimal React UI to list and add pets using the backend at `http://localhost:8080`.

## Dev Requirements

- Node.js 18+ recommended

## Run Dev Server

```bash
cd frontend
npm install
npm run dev
```

Vite dev server runs on `http://localhost:5173` and proxies API requests (`/api/*`) to `http://localhost:8080`.

## Build

```bash
npm run build
```

Built assets output to `frontend/dist`.
